# Akorith 0.6.0

Akorith 0.6.0 completes the chat-first product transition: General Chat now behaves like a focused
model conversation, Workspace behaves like a project-scoped Codex task, and Loop remains a distinct
concurrent Goal system.

## Highlights

- Rich Markdown/GFM messages with tables, checklists, links, and modern code blocks.
- Durable image, PDF, document, archive, source, and other local-file attachments.
- Session-isolated streaming, Stop controls, navigation continuity, and queued follow-ups.
- Workspace Plan mode, `@` project-file mentions, real Changes review, Stage/Unstage, and elapsed
  task summaries.
- Task search, pinning, background-work indicators, and a polished responsive sidebar.
- Unified dark/light product styling across Chat, Workspace, Loop, Dashboard, Benchmark, Plugins,
  Settings, and Changes.
- User-installed CLI resolution so Electron does not shadow a newer signed-in Codex executable.

## Attachment safety

Akorith keeps managed attachment copies inside its own application-data directory. A message is
limited to 8 files, 16 MB per file, and 40 MB total. Original files are never modified or deleted.

## Validation

This release passed TypeScript and production builds, all repository verification scripts, a
production dependency audit, and live Electron tests covering General Chat, Workspace, Loop,
navigation while streaming, Queue, Plan no-write behavior, file mentions, diff review,
Stage/Unstage, task search/pinning, Markdown, PDF/image attachments, both themes, and responsive
layouts.

macOS artifacts remain unsigned and may require right-clicking the app and choosing **Open** on the
first launch.
